# Super Store Sales Dashboard

An interactive dashboard for analyzing store sales, displaying key performance indicators such as sales, profits, customers, and orders, with temporal analysis of sales and profits across years. The dashboard includes detailed segmentations by category, region, sector, and shipping method, as well as an interactive map showing the geographic distribution of sales.
The dashboard also features a Forecasting capability that enables the estimation of future sales and supports strategic decision-making based on accurate data.
The interface was designed with a modern, user-friendly style, providing a clear, comprehensive view that empowers decision-makers to analyze performance and improve operational and marketing planning.

The dashboard was implemented using Power BI to deliver the best interactive experience and professional visual presentation of data.
